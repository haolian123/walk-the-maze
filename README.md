# 走迷宫小游戏

## 简介

使用Java的swing模块编写的软件课程作业。

使用了简单工厂模式用于创造不同难度的迷宫，便于后续的增加更多难度的迷宫。

如果需要添加迷宫，在MazeFactory中添加即可。

## 迷宫存储形式

使用二维数组存储

示例

```java
protected   int[][] maze ={
            {2, 2, 2, 2, 2, 2, 2, 2, 2},
            {2, 0, 0, 1, 1, 1, 0, 1, 2},
            {2, 1, 0, 0, 0, 1, 0, 1, 2},
            {2, 1, 0, 1, 0, 0, 0, 1, 2},
            {2, 0, 0, 1, 0, 1, 1, 1, 2},
            {2, 0, 1, 1, 0, 0, 0, 0, 2},
            {2, 1, 1, 1, 1, 1, 1, 0, 2},
            {2, 2, 2, 2, 2, 2, 2, 2, 2}

    };
```

1. '0'表示可通过。
2. '1'表示障碍。
3. '2'表示边界。

## 运行

运行Main.java中的main函数。
